Soucis de nbsp

La plateforme ne permet pas d'edition du code html.

J'ai essaye plusieurs methodes d'insertion de nbsp en utilisant different encodage. Mais a chaque fois Tribune transforme en espace normal, ca ne sert donc a rien. Peux importe ce que je cree ou copy/paste, ce n'est pas garde dans le resultat final.

Il y a cependant une inclusion automatique (-> meme si je met un espace normal, c'est convertit en nbsp) de la plateforme sur certain nbsp (en jaune sur l'image) malheureusement il en manque beaucoup d'autre (en rouge)

Je ne sais pas si quelque chose peut etre remonte a Tribune a ce niveau...

J'espere que ce n'est pas simplement moi qui ai loupe un truc.

Pour les chiffres il y a la possibilite d'ecrire 3.000 au lieu de 3 000 si ca interesse.

Si voulez faire des tests, utilisez mon petit script JS a envoyer dans la console du navigateur pour surligner en jaune les vrai nbsp :

function highlightNbsp(node) {
    if (node.nodeType === 3) {
        const newText = node.nodeValue.replace(/\u00A0/g, '<span style="background-color: yellow;">&nbsp;</span>');
        if (newText !== node.nodeValue) {
            const span = document.createElement('span');
            span.innerHTML = newText;
            node.parentNode.replaceChild(span, node);
        }
    } else if (node.nodeType === 1 && node.nodeName !== "SCRIPT" && node.nodeName !== "STYLE") {
        for (let i = 0; i < node.childNodes.length; i++) {
            highlightNbsp(node.childNodes[i]);
        }
    }
}
highlightNbsp(document.body);
